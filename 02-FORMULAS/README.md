# Micro Project Number 2 - Formulas & Math Operators

## Question

Calculate row-level totals and a dataset-level aggregate using arithmetic formulas.

## Dataset

- One ID column
- At least 2 numeric input columns (e.g., Quantity, Unit_Price)
- One calculated column derived from them

## Method(Excel)

- Splited data set into separate columns per delimiter.
- Converted data range into excel table.
- Used product to calculate the total cost per order
- Used sum to calculate the aggregate total

## Results

- Total Order quantity - 118
- Total Unit Price - $1,170
- Total Order -$2,778
